Linux的简单使用

安装好要用的虚拟机

关闭防火墙及SELinux以及设置主机名

组成：Linux由硬件、内核（含文件系统）、Shell、应用程序四层组成，Shell作为交互接口，衔接用户指令与内核，最终通过硬件完成操作反馈结果。

shell命令提示符：

  • [root@localhost ~]#
  • root ：当前登录的账户名，root为管理员
  • localhost：主机名，默认为localhost
  • ~ ：该位置表示当前工作目录，~表示/root(普通账户 ~ :/home/普通账户名)
  • 提示符标志：#或$ ,表示账户身份，#表示root，$表示普通账户

命令的格式：主命令 参数 操作对象

修改密码

更改root：passwd

更改用户：pa

清除用户密码：passwd -d fox

重置root密码

1，重置系统，按e键显示内核启动参数

2，在Linux这行末尾空格输入rd.break，再按ctrl+x

3，挂载/sysroot

输入mount -o remount,rw /sysroot/

mount | grep sysroot

4,设置密码

chroot /sysroot/

echo 060419 | passwd –stdin root

5，强制重置(若不重置密码修改无效)

touch /.autorelabel

6，输入两次exit，等待一会可进入登陆页面

info：用途与man相似，用来查询命令的用法或者是文件的格式

date：用于显示或设置系统的时间与日期

date -参数 +咸宁市日期时间格式参数

常见参数

  • %Y：完整年份（例如：2020）

  • %m：月份（1~12）

  • %d：本月中的第几天

  • %H：小时（00~23）

  • %M：分钟（00~59）

  • %j：今年中的第几天

  • %X：相当于%H:%M:%S

  • %Z：显示时区

  • %A：星期几 (Sunday-Saturday)

  • %p：显示本地AM或PM

timedatectl：用于设置系统的时间

timedatectl status（查看系统时间与地区）

timedatectl set-timezone 时区（设置时区)

timedatectl set-time 时间（修改时间）

常见快捷键

  • tab：单词补全

  • ctrl+c：终止当前任务

  • ctrl+l（小写L）：清屏

  • ctrl+insert：复制

  • shift+insert：粘贴

  • ctrl+shift+=：放大字号

  • ctrl+-（减号）：缩小字号

  • ctrl+z：终止进程

从命令行管理文件

硬链接：硬连接就是指向原始文件inode的指针

查看方法:

ll 文件名

stat 文件名

建立硬链接

ln -参数 源文件 目标连接文件

硬连接建立后，删除源文件后数据依然可以访问，只有硬连接为0时，数据无法访问

注意：硬连接不能给目录建立

软连接：一种符号连接，类似于windows的“快捷方式”，仅包含所连接的路径名

格式：ln -s 源文件 目标连接名

软连接只是一种指向关系，删除源文件则数据不可访问(与硬连接不同)

常见命令行指令

文件与目录操作

1、 ls  列出目录内容  例如：ls -l （详细列表）、 ls -a （显示隐藏文件）

2、 cd  切换目录  例如：cd /root （切根目录）、 cd .. （返回上级）

3、mkdir  创建目录例如： mkdir test_dir （创建单目录）、 mkdir -p a/b/c （递归创建）

4、 rm  删除文件/目录  例如：rm test.txt （删文件）、 rm -rf test_dir （强制删目录）

 5、cp  复制文件/目录  例如：cp file1 /tmp/ （复制文件）、 cp -r dir1 /tmp/ （复制目录）

6、 mv  移动/重命名文件  例如：mv file1 file2 （重命名）、 mv file1 /tmp/ （移动）

vim编辑器基础操作

一、模式切换指令

1. 进入插入模式：

  •  i ：光标当前位置插入文本

  •  a ：光标下一个位置插入文本

  •  o ：当前行下方插新行并进入插入模式

  •  O ：当前行上方插新行并进入插入模式

2. 进入命令行模式： : （普通模式下输入，底部出现命令行）

3. 返回普通模式： Esc （插入/命令行模式下通用）

二、文本编辑指令（普通模式）

1. 字符/行删除： x （删除光标处字符）、 dd （剪切当前行， ndd 剪切n行）

2. 复制/粘贴： yy （复制当前行）、 p （光标下方粘贴）、 P （光标上方粘贴）

3. 撤销/重做： u （撤销上一步）、 Ctrl+r （重做撤销操作）

三、保存与退出指令（命令行模式）

1. 保存： :w （保存文件，不退出）、 :w 新文件名 （另存为新文件）

2. 退出： :q （无修改时退出）、 :q! （强制退出，不保存修改）

3. 保存并退出： :wq / :x （两者效果一致，为原笔记补充的高频指令）

四、行操作

 gg （跳至文件首行）、 G （跳至文件末行）
